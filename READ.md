# Football Score Prediction Using Machine Learning

## Overview

This project predicts football match scores using Machine Learning techniques and historical match statistics. The model analyzes features such as shots taken and ball possession percentages to estimate the number of goals scored by both home and away teams.

## Features

* Data preprocessing using Pandas
* Feature selection and engineering
* Machine Learning model training using Random Forest Regression
* Home and Away score prediction
* Model evaluation using Mean Absolute Error (MAE)
* Git and GitHub integration for version control

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Git
* GitHub

## Dataset Features

The dataset includes:

* Home Team
* Away Team
* Home Shots
* Away Shots
* Home Possession (%)
* Away Possession (%)
* Home Goals
* Away Goals

## Machine Learning Workflow

1. Load and preprocess football match data
2. Split dataset into training and testing sets
3. Train Random Forest Regression model
4. Evaluate model performance
5. Predict future football match scores

## Example Prediction

Input:

* Home Shots: 16
* Away Shots: 11
* Home Possession: 58%
* Away Possession: 42%

Predicted Score:

* Home Goals: 2
* Away Goals: 1
